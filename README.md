# СЦЕНАРИИ

### Позитивный сценарий через "Каталог курсов" для зарегистрированного пользователя:
1. Перейти на сайт Нетология
2. На главной странице выбрать меню "Каталог курсов"
3. В разделе программирование выбрать "Тестировщик ПО"
4. Перейти к форме записи нажав на кнопку "Записаться"
5. В поле "Имя" ввести имя зарегистрированного пользователя
6. В поле "Телефон" ввести валидный номер телефона зарегистрированного пользователя
7. Нажать кнопку "Записаться"
8. Ожидаемый результат: форма записи на курс отправляется

# ИНСТРУМЕНТЫ

1. IntelliJ IDEA - интегрированная среда разработки, понимает многие языки программирования и предлагает интеллектуальную помощь при написании кода
2. Gradle - система управления проектом. Позволяет подключать различные плагины и зависимости, структурирует проект
3. Selenide - фреймворк для тестирования веб-интерфейсов (проще писать код)
4. Allure - система генерации отчетов, позволяющая анализировать прохождение тестов
5. Faker - генерация пользовательских данных. При отсутствии доступа к базе данных генерирует фейковые данные (не нужно придумывать самим) 
6. Git, Github - хранение проекта, возможность вносить изменения и их отслеживать
7. Java - язык прграммирования для написания кода
8. Lombok - библиотека для сокращения кода. Позволяет уменьшить объем написанного, используя геттеры, сеттеры и прочие стандатрные операции
